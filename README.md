A premium, full-stack digital experience for a high-end Moroccan tailoring house. **Annakhil** blends traditional craftsmanship with modern technology, providing a seamless interface for bespoke fashion management and luxury shopping.

---

## ✨ Key Features

### 🏛️ Premium Frontend
- **Luxury UI/UX**: Crafted with a focus on high-end aesthetics using custom CSS, glassmorphism, and cinematic animations.
- **Dynamic 3D Effects**: Interactive layers and parallax scrolling for a depth-filled user experience.
- **Responsive Mastery**: Flawless performance across all devices, from mobile to ultra-wide displays.
- **Modern Typography**: Sophisticated pairing of *Playfair Display* and *Inter* for that distinct luxury feel.

### ⚙️ Powerful Backend
- **Custom Admin Dashboard**: Full control over products, orders, and site-wide featured content.
- **Secure Authentication**: Robust implementation using **JWT**, **Bcrypt**, and integrated **Google OAuth 2.0**.
- **Performance Optimized**: Built on **Express 5.x** with efficient MongoDB querying and middleware-driven architecture.
- **Advanced Security**: Protected by **Helmet**, **HPP**, Rate Limiting, and XSS sanitization.

## 🚀 Installation & Setup

To explore the codebase locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/cobra-x-ops/said-tailleur.git
   cd said-tailleur
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root and add your configurations (refer to `.env.example` if available).

4. **Run the application**:
   - Start the server:
     ```bash
     npm start
     ```
   - (Optional) Start the frontend server separately if needed:
     ```bash
     npm run client
     ```
